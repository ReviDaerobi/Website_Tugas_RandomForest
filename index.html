Apa Arti Data Tersebut?

Glucose: Kadar gula dalam darah (Indikator Utama).

BMI: Indeks Massa Tubuh (Mengukur obesitas).

BloodPressure: Tekanan darah diastolik (mm Hg).

Age: Umur pasien.

Outcome: Target Prediksi (0 = Sehat, 1 = Diabetes).

Metode: Random Forest

Konsep: "Ensemble Learning"

Menggunakan 100 "Pohon Keputusan" (Trees) yang melakukan voting untuk menentukan hasil akhir. Lebih akurat daripada memprediksi dengan satu pohon saja.

*Dari mana angka ini berasal? Lihat slide selanjutnya.

1. Bedah Perhitungan: Confusion Matrix

Tabel perbandingan Tebakan Komputer vs Fakta Asli:

Tebakan: SEHAT (0) Tebakan: SAKIT (1)
Fakta: SEHAT (0) 85 (Benar) 14 (Salah Tuduh)
Fakta: SAKIT (1) 18 (Lolos) 37 (Benar)

Total Data Uji: 85 + 14 + 18 + 37 = 154 Data

2. Menghitung Precision

"Seberapa akurat saat komputer memvonis Sakit?"

Rumus = Tebakan Benar Sakit / Total Tuduhan Sakit

= 37 / (37 + 14)

= 37 / 51

= 0.72 (72%)

Artinya: Saat model bilang "Diabetes", kemungkinannya 72% benar.

3. Menghitung Recall

"Seberapa banyak pasien Sakit yang berhasil ditemukan?"

Rumus = Tebakan Benar Sakit / Total Orang Sakit Asli

= 37 / (37 + 18)

= 37 / 55

= 0.67 (67%)

Artinya: Model berhasil mendeteksi 67% dari total penderita.

4. Menghitung Akurasi

"Berapa total tebakan yang benar secara keseluruhan?"

Rumus = (Benar Sehat + Benar Sakit) / Total Data

= (85 + 37) / 154

= 122 / 154

= 0.79 (79%)

Artinya: Secara umum, nilai ujian model ini adalah 79.

Faktor Paling Berpengaruh

Berdasarkan analisis algoritma, berikut adalah variabel penentu utama:

Insight: Menjaga kadar gula dan berat badan adalah kunci pencegahan diabetes.
